Project management Tool

The ProjectManagement tool is a collaborative project management application built using Django and SQLite. It allows users to create projects and manage them by adding and updating tasks. Tasks can be assigned to team members, and their progress can be tracked with status updates. User permissions are implemented to ensure that only project members have access to view and modify tasks within a project, maintaining security and confidentiality. This application facilitates efficient project management by providing structured workflows and access controls, enhancing team collaboration and productivity.
